jQuery YouTube Plugin

Embed YouTube videos using YouTubes iframe or object/embed methods.

SwfObject is required for the object/embed method. SwfObject is not required for iframe method.

By defalut the iframe method is used.

Download

Download from Github.

Options

There are 9 options (default values shown).

                iframe: true,
                videoID: "",        // required
                width: "480",       // required
                height: "390",      // required
                play: "0",
                loop: "0",          // object/embed only
                link: "external",   // object/embed only
                controls: "1",      // iframe only
                autohide: "1"
